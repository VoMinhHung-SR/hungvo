# PROJECT BRIEF

Build a modern personal portfolio website for a Frontend Developer.

DO NOT copy either website directly.

The goal is to create an original portfolio inspired by their strengths.

---

# DESIGN DIRECTION

Use:

* Clean vertical layout
* Excellent spacing
* Minimal navigation
* Strong content hierarchy
* Section-based storytelling
* Professional and readable structure

Avoid:

* Large amounts of text
* Long biography sections
* Resume-like experience sections

---

Use:

* Dark premium theme
* Subtle glow effects
* Technical/developer atmosphere
* Modern typography
* Minimal visual decorations
* Clean developer identity

Avoid:

* Excessive animations
* Busy backgrounds
* Too many social links
* Visual clutter

---

# TARGET AUDIENCE

Recruiters
Software Engineers
Frontend Developers
Potential Clients

The portfolio should feel:

* Professional
* Technical
* Minimal
* Modern
* Intentional

Not flashy.
